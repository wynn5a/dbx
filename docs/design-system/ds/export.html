# Data Buddy — Design System (v1.1)

> Single-file reference for coding agents. Everything needed to build UI
> that looks like Data Buddy: tokens, type scale, component recipes, state rules.

## Identity
A dense, calm, keyboard-first dark interface in the Linear lineage.
Hierarchy from elevation + a 4-step text ramp — never heavy borders or big type.
Color means something: neutral by default; accent = action, selection, focus.
Geist for chrome, Geist Mono for anything machine-shaped.
