Typography

Two families do all the work. Geist for the interface; Geist Mono for anything that is data — IDs, numbers, timestamps, SQL, type names, keys. Numerals are always tabular so columns align.

Working rules
Tabular numerals

font-variant-numeric: tabular-nums on every grid and result table so digits sit in fixed columns.

Uppercase labels

Group headers are 11px / 600, uppercase, letter-spacing 0.05em, in tertiary text.

Mono for data

If a value is machine-shaped — id, count, ms, hex, SQL — it is set in Geist Mono, never the UI font.

Negative tracking up top

Display sizes (16px+) tighten to −0.01em; body and mono stay at default tracking.
