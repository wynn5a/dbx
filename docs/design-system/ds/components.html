Components

The working vocabulary. Borderless surfaces, hairline dividers, and one accent for primary actions and focus. Hover lifts the background by a hair; active state fills it slightly more. Icons are 1.5px line icons sized 11–17px.

Buttons

Primary uses a vertical accent gradient with a top sheen and 1px luminous border — a soft “lit from above” read. Secondary “chips” are hairline-bordered and ghost-filled; the active variant fills with --bg-active. Destructive borrows the red border.

Icon buttons

28px square, 6px radius, transparent until hover. Used for refresh, export, navigation, and close. Tertiary text resolves to primary on hover.

Navigation rows

Active row fills with --bg-active, text goes primary, leading icon goes accent. Idle is secondary text; hover fills faintly.
